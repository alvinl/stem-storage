Stem storage
===================

### Installation

1. `npm install git://github.com/alvinl/stem-storage.git`
2. Add `"stem-storage": {}` to plugins.json

### Config

```json
{

  "stem-storage": {
    
    "alertAdmins": false

  }

}

```

- `alertAdmins` Send a message to all admins when a trade offer is accepted or denied (defaults to false)
